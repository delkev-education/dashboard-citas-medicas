# Dashboard de Gestión de Citas Médicas

Proyecto desarrollado con Python y Streamlit para visualizar información de citas médicas.

## Tecnologías

- Python
- Pandas
- Streamlit
- Plotly
- Faker
- Scikit-Learn

## Archivos

- app.py
- citas_medicas.csv
- requirements.txt

## Ejecutar

Instalar dependencias:

pip install -r requirements.txt

Ejecutar la aplicación:

streamlit run app.py
